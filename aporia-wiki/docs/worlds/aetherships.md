# Aetherships

`world_id`: `aetherships`

A gaslamp-fantasy sky where flight itself runs on a finite fuel — raw aether-crystal, formed only where lightning strikes cloud for centuries. Eight standing fleets contest the same shrinking sky under an uneasy peace-through-exhaustion, none able to defeat all the others. Eight Factions. One Sky. No Quarter.
